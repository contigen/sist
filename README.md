# Sist

Agent Deployment Playground — build, configure, and deploy conversational agents without writing backend code.

## Features

- **Agent Builder** — Define purpose, capabilities, constraints, and system instructions
- **Model Selection** — Gemini and Claude model families with configurable parameters
- **Tools** — Web search, code execution, image generation, URL fetching, retrieval
- **One-Click Deploy** — Agents expose a streaming chat API endpoint
- **Preview Sandbox** — Test agents with real-time logs and tool call visibility

## Stack

Next.js 15 · Prisma · PostgreSQL · Vercel AI SDK · NextAuth · Tailwind CSS
